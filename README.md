# Django in Production - From Zero to Hero

Pretty good start for ansible structure  to deploy a django application.

Usage
=======

    python -m venv .venv
    source .venv/bin/activate
    pip install ansible
    ansible-playbook ansible/deploy/deploy-demo.yml -i ansible/hosts
